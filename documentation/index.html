mb-manage-ui documentation

MB Manage Pvt. Ltd.

An Online portal for colleges

Tech Stack

  • Spring boot 2.1.0

  • Angular 7.0.3

Steps to run

  1. Go to mb-manage-service and fire mvn spring-boot:run
  2. Go to mb-manage-ui and run npm install and then fire ng serve on the same path
  3. Populate Data by hitting endpoint http://localhost:8080/mb-manage-service/api/v1/commons/populate
  4. Go to http://localhost:4200
